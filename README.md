# Non_desisto

Wargame Study Repository
Overview

This repository documents my wargame problem-solving process with a focus on understanding vulnerabilities, not just finding solutions.

The goal is to build a reproducible methodology for analyzing and exploiting common security flaws.

Objectives
Develop a structured approach to vulnerability analysis
Reproduce exploits without relying on write-ups
Generalize attack patterns across different problems
